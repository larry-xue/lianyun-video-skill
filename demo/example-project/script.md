# 《大模型里的 token》3-slide 脚本

总时长：约 15 秒（5s + 5s + 6s，转场重叠后 14.8s）

## Slide 1 · Hook · 5s

**画面**：一句“你好”，旁边不是眼睛，而是一排编号 token。

**口播**：你发给大模型一句“你好”，它真的看见这两个字了吗？其实，它先看到的是 token。

## Slide 2 · 类比 · 5s

**画面**：整张披萨被切成大小不同的块，每块对应一个文本片段。

**口播**：把一句话想成披萨。tokenizer 先切块，每一块就是一个 token；切法还会因模型而不同。

## Slide 3 · 为什么重要 · 6s

**画面**：文本片段变成编号，流入“上下文窗口”容量条。

**口播**：模型处理的是这些编号。它们占多少空间，决定上下文能装多少，也常常影响 API 计费。

## 去 AI 腔自检

已按 skill 内的 humanizer 标准人工过稿：删掉“简单来说 / 本质就是 / 你必须记住”等讲课腔；保留对话式提问、披萨类比和“因模型而不同”的边界。
