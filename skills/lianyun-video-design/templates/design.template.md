## 一、灵魂(叙事 DNA)

- **你是谁**:[一句话。_例:你是炼云,一个独立 AI 开发者,不是在念 PPT,是在跟朋友解释一个概念。_]
- **你和观众的关系**:[_例:平等。不是居高临下地教,而是「我搞懂了个东西,来给你讲讲」。_]
- **怎么说话**:[_例:第二人称对话;口语但不碎;爱用类比;关键转折用短句。_]
- **一句反例对照**:[_例:❌「记住这三条」→ ✅「三个角度,咱们一个个过」。_]

> 完整的写作态度硬标准在 `lianyun-video-content`(把观众当朋友 / 不盖棺定论 / 埋 hook / 保留不确定)。

## 二、品牌 token(落到 `src/theme.ts`)

```
配色:
  primary        [#______]   主色:强调块 / eyebrow
  accent         [#______]   强调色:数字 / 高亮 / kicker
  text           [#______]   正文
  textSecondary  [#______]   注释 / 副标题 / footer
  bg             [#______]   背景(整片统一,建立辨识度)

字体:
  body  [_例:Noto Sans SC_]   正文、标题
  code  [_例:JetBrains Mono_] eyebrow、数字、meta、footer

品牌:
  handle   [@______]   右上角
  series   [______]    顶栏系列名
  footer   [______]    底栏
```

> **一套视觉 DNA**:观众看到任何一张 slide 都能一眼认出「这是你」。一个主色贯穿全系列。

## 三、frame chrome + 安全区

- **画布 / 帧率**:[_例:1920×1080 @ 30fps,16:9 横屏_]
- **内容安全区**:[_例:top 140 / bottom 180 / left 72 / 右边界 x=1580_]
- **平台互动区**:[_例:抖音横屏观看,右侧 340px 被点赞/评论覆盖,不放可读内容_]
- **四角固定 chrome**:[_例:左上 section eyebrow、右上 brand tag、左下 footer、右下页码_]

> 细则见 `references/frame-and-density.md`。

## 四、slide 版式 + 动画

- **常用版式**:[_例:标题页 / 要点列表 / 左右对比 / 数据柱 / 代码讲解_]
- **转场**:[_例:统一 slide from-bottom,禁装饰转场_]
- **逐步揭示**:[_例:多要点跟节奏逐条出现,不一次全 stagger_]
- **何时持续动**:[_例:出现「量 / 占比 / 增长 / 流程」就让它动起来,否则静止_]

> 细则见 `references/slides-and-motion.md`。

## 五、密度与 QA 硬标准

- **字号**:[_例:主标题 56–96px、secondary 26–38px、card padding 28–36px_]
